### Title
Missing Group Ownership Check in `removeTransactionGroup` Enables Permanent Hard-Deletion of Any Transaction Group's Items by Any Authenticated User

### Summary
The `DELETE /transaction-groups/:id` endpoint does not verify that the requesting user owns the target transaction group. Inside `removeTransactionGroup`, each `TransactionGroupItem` is **hard-deleted** from the database before the per-transaction creator check runs. This means any authenticated, verified user can permanently corrupt the structure of any transaction group they do not own, with no recovery path.

### Finding Description

**Root cause:** `removeTransactionGroup` in `transaction-groups.service.ts` fetches the group by ID with no ownership assertion, then iterates its items and issues a hard `remove` on each `TransactionGroupItem` *before* delegating to `removeTransaction`, which is where the creator check actually lives.

```
DELETE /transaction-groups/:id
  → TransactionGroupsController.removeTransactionGroup   (no AdminGuard; any VerifiedUser)
  → TransactionGroupsService.removeTransactionGroup      (no group-ownership check)
      for each groupItem:
          dataSource.manager.remove(TransactionGroupItem, groupItem)   ← HARD DELETE, permanent
          transactionsService.removeTransaction(transactionId, user, false)  ← throws if not creator

  async removeTransactionGroup(user: User, id: number): Promise<boolean> {
    const group = await this.dataSource.manager.findOneBy(TransactionGroup, { id });
    if (!group) {
      throw new Error('group not found');
    }
    const groupItems = await this.dataSource.manager.find(TransactionGroupItem, {
      relations: {
        group: true,
      },
      where: {
        group: {
          id: group.id,
        },
      },
    });
    for (const groupItem of groupItems) {
      const transactionId = groupItem.transactionId;
      await this.dataSource.manager.remove(TransactionGroupItem, groupItem);
      await this.transactionsService.removeTransaction(transactionId, user, false);
    }

    await this.dataSource.manager.remove(TransactionGroup, group);

    emitTransactionUpdate(this.notificationsPublisher, groupItems.map(gi => ({ entityId: gi.transactionId })));

    return true;
  }
