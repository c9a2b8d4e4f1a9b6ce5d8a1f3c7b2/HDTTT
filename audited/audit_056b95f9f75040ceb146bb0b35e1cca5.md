### Title
Child Approver Removal Does Not Adjust Parent Threshold, Permanently Blocking Transaction Approval

### Summary
The `removeTransactionApprover` function removes a child approver node from a threshold tree without adjusting the parent's threshold. This can leave the parent with a threshold that exceeds the number of remaining children, making the approval condition permanently unsatisfiable and the transaction permanently stuck in `WAITING_FOR_SIGNATURES`.
